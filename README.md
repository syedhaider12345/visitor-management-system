# Visitor Management System 🚪✨

A simple, easy-to-use **Visitor Management System** built with [Streamlit](https://streamlit.io/) and SQLite.

## 📌 Features
- ✅ Visitor Check-In with photo capture
- ✅ Secure Check-Out with gate pass
- ✅ Admin Dashboard to view and download all visitor records

## 🚀 Getting Started

1️⃣ **Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/visitor-management-system.git
cd visitor-management-system
```

2️⃣ **Install dependencies**
```bash
pip install -r requirements.txt
```

3️⃣ **Run the app**
```bash
streamlit run app.py
```
